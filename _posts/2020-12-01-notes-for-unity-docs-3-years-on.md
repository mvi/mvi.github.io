# Notes for Unity Docs, 3 years on

<img src="/Images/NotesForUnity3YearsOn.png" />

It’s been a little over 3 years since I created a Chrome extension for shared notes on Unity Docs pages. In that time 465 notes have been posted by 111 different users! And every single note adds something meaningful to those pages, whether it’s a usage example, undocumented gotcha, example output, or a detailed writeup on a topic.

The notes are moderated by myself and my good friend [Thom Ingram](https://vertx.xyz/), but we’ve been fortunate that very little moderation needs to take place. The quality of the notes in particular are very high and it’s a fantastic feeling when working on a problem to load up the doc page and see a helpful comment that saves you hours of time.

Thank you to everyone who has posted a note or downloaded the extension. I’m also thankful for the lovely Unity Docs team for their support and I’m looking forward to reaching 500 notes soon!

If you’re a Chrome user and haven’t got the Chrome extension already you can download it [here](https://chrome.google.com/webstore/detail/user-contributed-notes-fo/fchdfdnnpkphopmdaochdfnmcahndmnb).
